#webdev-initial
npm scriptのみで構成した低依存のWebプロジェクトビルドツールです。

## Spec

* PostCSS
  * postcss-import
  * postcss-extend
  * postcss-mixins
  * postcss-comment
  * postcss-nested
  * postcss-color-function
  * postcss-simple-vars
  * autoprefixer
  * cssnano
  * cssnext
  * precss
* Babel(ES2015)
* eslint
* watchify
* browserify
* watchify
* Browsersync

## Command

  $ npm run start
